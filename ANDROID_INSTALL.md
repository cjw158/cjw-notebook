# 在 Android 设备上安装和使用

## 安装方法

### 方法 1: 通过 Chrome 浏览器安装（推荐）

1. 在 Android 设备上打开 Chrome 浏览器
2. 访问应用的网址
3. 点击右上角的菜单按钮（三个点）
4. 选择"添加到主屏幕"或"安装应用"
5. 输入应用名称（默认为"MindSpace"）
6. 点击"添加"
7. 应用图标将出现在您的主屏幕上

### 方法 2: 通过其他浏览器

某些浏览器（如 Firefox、Edge）也支持 PWA 安装：
1. 访问应用网址
2. 查找浏览器提供的"添加到主屏幕"选项
3. 按照提示完成安装

## 功能特性

### 📝 笔记管理
- **创建笔记**: 点击右下角的蓝色"+"按钮
- **编辑笔记**: 点击侧边栏中的任何笔记即可编辑
- **Markdown 支持**: 点击工具栏中的眼睛图标预览 Markdown 格式
- **收藏笔记**: 点击星标图标标记重要笔记
- **搜索笔记**: 使用顶部搜索框快速查找

### ✅ 待办事项
- **切换到待办**: 点击顶部的"待办"标签
- **添加任务**: 在输入框中输入任务，按回车或点击"添加"按钮
- **完成任务**: 点击任务前的复选框标记为已完成
- **删除任务**: 悬停或长按任务，点击垃圾桶图标
- **清除已完成**: 滚动到列表底部，点击"清除已完成"按钮

### 🎨 主题和外观
- **深色模式**: 点击侧边栏顶部的月亮/太阳图标切换主题
- **自动适配**: 应用会记住您的主题偏好

### 📤 导出功能
- **单个笔记**: 点击工具栏中的下载图标，选择 TXT 或 PDF 格式
- **所有笔记**: 点击侧边栏中的"⋮"菜单，选择批量导出选项

### 🔄 排序
- 点击侧边栏中的排序图标
- 可按修改时间、创建时间或标题排序

## 离线使用

应用支持离线使用：
- 首次访问后，应用会缓存必要的文件
- 即使没有网络连接，您也可以继续使用
- 所有数据都保存在您的设备上，不会发送到服务器

## 数据存储

- 所有笔记和待办事项都存储在浏览器的本地存储中
- 数据不会同步到其他设备
- 清除浏览器数据会删除所有笔记和待办事项
- 建议定期使用导出功能备份重要数据

## 卸载应用

### Android 设备
1. 长按应用图标
2. 选择"卸载"或"从主屏幕移除"
3. 或者：进入设置 > 应用 > MindSpace > 卸载

## 故障排除

### 应用无法安装
- 确保使用 Chrome 浏览器（版本 56+）
- 检查浏览器是否启用了 JavaScript
- 尝试清除浏览器缓存后重新访问

### 离线模式不工作
- 确保至少访问过应用一次并完全加载
- 检查浏览器是否允许网站存储数据
- 在 Chrome 中：设置 > 网站设置 > 存储空间

### 数据丢失
- 不要清除浏览器数据
- 使用导出功能定期备份
- 如果意外清除了数据，无法恢复（数据仅存储在本地）

## 系统要求

- Android 5.0 或更高版本
- Chrome 56+ 或其他支持 PWA 的现代浏览器
- 至少 10MB 可用存储空间
